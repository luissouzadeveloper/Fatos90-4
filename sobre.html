Sobre o site

Primeiramente, muito bem vindo! A Fatos 90 é um site que aborda acontecimentos para entusiastas dessa década com momentos bons e ruins. O site começou com uma iniciativa educacional lá na versão 1, e eu como autor hospedei o site para consolidar meus conhecimentos em desenvolvimeto web e acabei me divertindo muito, tanto, que já conta com mais 3 versões.

Quem sou eu

Eu me chamo Luis Felipe, sou estudante de desenvolvimento web e criador desse site. Eu tenho o objetivo de trabalhar na área front end, criando sites profissionais. Esse site tem me dado pequenas experiências em construções de páginas na web.

Tecnologias utilizadas

A três tecnologias que estão na imagem acima foram utilizadas no site. O HTML 5 é uma linguagem de marcação onde é possível manipular os textos, títulos, imagens e etc. O CSS 3 é uma linguagem de folhas de estilos, ou seja, serve para estilizar o HMTL 5 para ganhar mais estética e ficar um site mais atraente. E por último, temos o Bootstrap que é a novidade do site. Essa tecnologia é uma framework, ou seja, uma concentração enorme de códigos CSS prontos que pode ser utilizado com mais facilidade, o bootstrap foca também na responsividade.
